# Logistics Modules Docs

This folder documents logistics modules based on the current project structure.

## Modules

- `shipment.md` - Shipment and stock movement tracking (`transport_tracking` domain)
- `fleet.md` - Truck, driver, transporter, and maintenance operations (`trucks` domain)
- `warehouse.md` - Planned warehouse scope and integration notes

## How to Use

1. Update the relevant module file when adding or changing behavior.
2. Add payroll impact notes if shared components are touched.
3. Keep route names and controller references aligned with actual code.
